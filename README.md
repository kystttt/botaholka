# Telegram/Discord бот для ресторана

### Возможности бота

У покупателя есть возможность просмотра списка текущих заказов(своих), просмотра меню, добавления позиции в корзину, отмены заказа, повтора заказа, удаления позиции из корзины, сборки заказа из корзины, просмотра истории заказов, а также отзывов и возможности добавить отзыв после заказа.

У продавца есть возможность просмотра просмотра всех заказов, просмотра и изменения конкретного(на случай, если какая-то позиция заказа закончится), а также изменение статуса конкретного заказа, например, был "готовится", а после изменения стал "готов".


## Пример импользования
### Для клиента:
    /listoforders №Заказа - просмотр текущих заказов
    /duplicate №Заказа - 	повторить заказ
    /cancel №Заказа - отменить заказ
    /viewcart - просмотр корзины
    /menu - вызов меню(с последующим ввода пунктов меню, для их добавления в корзину)
    /makeOrder - создает заказ из того, что в корзине
    /deletefromcart - удаляет позицию в корзине по ее индексу
    /history - история заказов
    /allreviews - посмотреть все отзывы
    /next - просмотр следующих 5 отзывов
    Функция оставления отзыва доступна после заказа, для этого сначала надо написать свой отзыв и после этого поставить ему оценку
    При просмотре всех отзывов(/allreviews) можно смотреть не больше 5 отзывов за раз, для просмотра следующих отзывов надо вызывать команду /next


### Для продавца:
    /orders - просмотр всех текущих заказов
    /order №Заказа - просмотр и изменение конкретного заказа
    /nextStatus - изменение статуса заказа на следующий (т.е. например из готовится в приготовлен)
